# Product

## Register

product

## Users

นักศึกษาและบุคคลทั่วไปที่ต้องการเรียนรู้กฎหมาย IT เบื้องต้น เช่น พ.ร.บ.คอมพิวเตอร์, PDPA, ลิขสิทธิ์ออนไลน์ ผู้ใช้เข้ามาเพื่อหาข้อมูลอ้างอิงอย่างรวดเร็ว บนมือถือหรือคอมพิวเตอร์ ในบริบทการเรียนหรือการทำงาน ต้องการคำตอบที่ชัดเจนโดยไม่ต้องอ่านกฎหมายฉบับเต็ม

## Product Purpose

ศูนย์รวมกฎหมายด้านเทคโนโลยีสารสนเทศของไทย นำเสนอข้อมูลกฎหมายฉบับสรุปที่เข้าใจง่าย เพื่อการศึกษาและการใช้งานเบื้องต้น ความสำเร็จคือเมื่อผู้ใช้เข้ามาแล้วพบข้อมูลที่ต้องการภายใน 30 วินาที เข้าใจได้ทันที และรู้สึกว่าเว็บนี้น่าเชื่อถือ

## Brand Personality

ทันสมัย สะอาด เข้าถึงง่าย

เว็บนี้ควรให้ความรู้สึกว่ากฎหมายไม่ใช่เรื่องน่ากลัวหรือซับซ้อน ด้วยน้ำเสียงที่มั่นใจแต่ไม่น่ากลัว สบายตาแต่ไม่เรียบจนน่าเบื่อ

**References:** Linear, Notion — สะอาด เรียบง่าย ใช้งานได้ทันทีโดยไม่ต้องคิดมาก

## Anti-references

- เว็บราชการไทยแบบเก่า: เลย์เอาท์ล้าสมัย สีเทาจืดๆ ตารางซ้ำซ้อน ฟอนต์เล็ก อ่านยาก ไม่ responsive
- เว็บที่ดูยุ่งเหยิงเกินไป: เต็มไปหมด ไม่รู้จะเริ่มตรงไหน
- เว็บ Wikipedia-style ที่เป็นข้อความล้วน ไม่มีโครงสร้าง visual ที่ช่วยนำทาง

## Design Principles

1. **ชัดเจนเป็นอันดับหนึ่ง** — ทุกหน้าต้องบอกได้ทันทีว่าผู้ใช้อยู่ตรงไหน และจะหาอะไรได้อย่างไร
2. **สรุปก่อน ขยายทีหลัง** — แสดงใจความหลักก่อน ให้ผู้ใช้เลือกเจาะลึกเองเมื่อต้องการ
3. **ทันสมัยแต่จริงจัง** — ดีไซน์ที่ดีสร้างความน่าเชื่อถือ ไม่ใช่ดูเล่นๆ แต่ก็ไม่ดูแข็งทื่อ
4. **เร็วทุกจุดสัมผัส** — จากคลิกถึงคำตอบต้องสั้นที่สุด ทั้งเวลาโหลดและจำนวนขั้นตอน
5. **ให้ทุกคนเข้าถึงได้** — ข้อมูลกฎหมายเป็นเรื่องของทุกคน อินเทอร์เฟซต้องไม่เป็นอุปสรรค

## Accessibility & Inclusion

- WCAG 2.1 Level AA
- Contrast ratio ≥ 4.5:1 สำหรับ body text, ≥ 3:1 สำหรับ large text
- รองรับ keyboard navigation ครบทุกหน้า
- Semantic HTML สำหรับ screen reader
- รองรับ reduced motion preference
- ฟอนต์ที่อ่านง่ายสำหรับภาษาไทย ขนาดไม่ต่ำกว่า 16px สำหรับ body text
